负二进制转换

给你一个整数 n ，以二进制字符串的形式返回该整数的 负二进制（base -2）表示。
注意，除非字符串就是 “0”，否则返回的字符串中不能含有前导零。

负二进制数相加

给出基数为 -2 的两个数 arr1 和 arr2，返回两数相加的结果。

数字以 数组形式 给出：数组由若干 0 和 1 组成，按最高有效位到最低有效位的顺序排列。例如，arr = [1,1,0,1] 表示数字 (-2)^3 + (-2)^2 + (-2)^0 = -3。数组形式 中的数字 arr 也同样不含前导零：即 arr == [0] 或 arr[0] == 1。

返回相同表示形式的 arr1 和 arr2 相加的结果。两数的表示形式为：不含前导零、由若干 0 和 1 组成的数组。

  • 相加时第i位发生进位时，carry置为-1
  • arr1[i] -> [0, 1] arr2[j] -> [0, 1] carry -> [-1, 0] 因此相加时每一位三值和 x = arr1[i] + arr2[j] + carry 为 [-1, 0, 1, 2]其中之一
    • x == 0, 1 时，答案数组中该位直接为x的值，carry置0
    • x == 2时，答案数组中该位为 x % 2，carry置-1
    • x == -1时，答案数组中该位为1，carry置1，因为：$-(-2)^i=(-2)^{i+1}+(-2)^i$

  • c == 2时，答案数组中该位为 c % 2，c置-1
